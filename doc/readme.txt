依赖包：
pyside6
pandas
openpyxl

界面：
...（按钮）：选择文件窗口
分析：解析选择的.xlsx文件中的账号类型，并显示；自动读取accountname字段，缺省该字段读取第一列作为账号
筛选：输出选择的账号信息，并附加账号类型
附加：输出所有账号信息，并附加账号类型

账号类型：
char_reg：纯字母组合，类型二为长度
char_num_reg：字母+数字组合，类型二为长度
mail_reg：邮箱组合，类型二为邮箱网站
none_type：其他组合，类型二为组合结构，c表示小写字母，C表示大写字母，n表示数字，f表示符号
